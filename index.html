📘 Como usar o Livro Caixa

Seu caderninho digital para organizar o dinheiro das lojas Be Make! ✨

🎯 O que é o Livro Caixa?

Uma ferramenta SIMPLES para registrar todo o movimento de dinheiro do dia. É como um "caderninho digital" que faz as contas sozinho!

⭐ Para que serve?

  • 📝 Registrar entradas e saídas de dinheiro
  • 🧮 Calcular totais automaticamente (cartão, PIX, entregas)
  • 📊 Gerar relatórios em PDF profissionais
  • 👥 Acompanhar desempenho das vendedoras
  • 💾 Salvar automaticamente na nuvem

🚀 Como usar - Passo a Passo Simples:

1 Acesse o Livro Caixa

Clique no botão "Acessar Livro Caixa" acima

2 Preencha as informações básicas

• Data (já vem com hoje)
• Loja (Norte, Centro, etc)
• Seu nome

3 Registre o dinheiro do caixa

• Valor inicial em dinheiro
• Valor final em dinheiro
• O sistema calcula a diferença sozinho!

4 Adicione as vendas

Preencha quanto entrou por:
💳 Cartão Débito | 💳 Cartão Crédito
📱 PIX | 🔗 Link Pagamento | 🎫 Vale Presente

5 Movimentações extras (opcional)

• 🚚 Entregas feitas
• 💸 Retiradas do caixa
• 🛍️ Compras do dia
• 👩‍💼 Tickets das vendedoras

6 Finalize o dia

Clique em "Enviar Livro Caixa"
O sistema gera um PDF automático com todo resumo!

💡 Dicas Importantes:

  • ✅ Preencha todo dia - Não deixe acumular
  • ✅ Use números com ponto - Ex: 100.50 (não use vírgula)
  • ✅ Revise antes de enviar - Confira se está tudo certo
  • ✅ Salve o PDF - É seu comprovante do dia

🎉 Pronto!

Em menos de 5 minutos você tem todo controle do caixa organizado e profissional!

🔔 Novidades e Atualizações

🎉 Confira as últimas melhorias do sistema BE MAKE!

✨ Campo de Observações no Formulário

Agora você pode adicionar observações personalizadas ao preencher o Livro Caixa! Isso ajuda a registrar informações importantes como:

  • 📝 Detalhes sobre vendas especiais
  • 💰 Observações sobre valores diferenciados
  • 🗒️ Notas sobre o movimento do dia
  • 📋 Informações adicionais para o financeiro

📄 Campo de Observações no PDF

As observações que você inserir no formulário agora aparecem automaticamente no PDF gerado! Mantenha todo o histórico organizado e profissional.

🕐 Data com Timezone GMT-3 (Brasil)

A data e hora agora são registradas automaticamente no horário oficial do Brasil (GMT-3). Não importa onde você esteja, o sistema sempre mostrará a hora certa!

🔗 Novo Link do Livro Caixa

O sistema foi atualizado com o novo link do Livro Caixa para melhor desempenho e estabilidade.

🎯 Benefícios das Atualizações:

  • ✅ PDF completo - Todas as informações em um só documento
  • ✅ Hora precisa - Timezone brasileiro para evitar confusões
  • ✅ Melhor performance - Sistema otimizado e mais rápido
